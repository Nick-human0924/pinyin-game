# 拼音大冒险 v8.2.0 更新日志

**发布日期**: 2026-03-06

## 🎮 新增功能

### 1. 关卡地图重构
- **按阶段分组显示**: 关卡现在按照学习进度分为5个阶段
  - 第一阶段: 声母 (1-3关)
  - 第二阶段: 韵母 (4-6关)
  - 第三阶段: 整体认读 (7-9关)
  - 第四阶段: 声调 (10-12关)
  - 第五阶段: 词语 (13-15关)
- **阶段标题和进度**: 每个阶段显示图标、标题和完成进度
- **关卡描述**: 每个关卡显示类型和描述信息

### 2. 错题本系统
- **错题记录**: 自动记录答错的题目
- **错题查看**: 查看错题详情，包括正确答案和错误答案
- **错题复习**: 支持针对性复习错题，检验掌握情况
- **掌握标记**: 已掌握的题目可以标记移除
- **统计展示**: 显示错题总数和已订正数量

### 3. 宠物商店
- **宠物分类**: 商店新增宠物专属分类
- **多种宠物可选**:
  - 🐼 拼音小熊猫 (默认): 平衡型
  - 🐯 勇猛小老虎: 高攻击型 (攻18 防3 血85)
  - 🐰 迅捷小兔: 敏捷型 (攻8 防6 血80)
  - 🦁 威武小狮子: 攻守平衡型 (攻15 防8 血95)
  - 🦄 魔法独角兽: 高防御型 (攻12 防15 血110)
  - 🐲 火焰小龙: 强力攻击型 (攻25 防5 血90)
- **宠物切换**: 可以在"我的宠物"页面切换出战的宠物
- **不同属性**: 每种宠物有不同的攻击、防御、血量属性

### 4. 背包系统
- **背包页面**: 新增独立的背包界面
- **宝物分类**: 
  - 💎 宝物 (用于装备升级)
  - 🔧 材料 (预留)
  - 🧪 消耗品 (预留)
- **探险掉落**: 野外探险击败怪物后有几率掉落宝物

### 5. 装备升级系统
- **装备管理**: 新增装备升级页面
- **三件装备**:
  - ⚔️ 武器: 影响攻击力
  - 🛡️ 防具: 影响防御力
  - 💍 饰品: 平衡加成
- **升级机制**: 
  - 武器升级需要红宝石
  - 防具升级需要蓝宝石
  - 饰品升级需要绿宝石
  - 每次升级需要消耗金币
- **属性提升**: 升级后装备属性提升，角色能力增强

## 🐛 修复与优化

- 优化了宠物页面的展示效果
- 探险战斗现在会掉落宝石材料
- 改进了错题记录的准确性
- 数据保存支持新系统

# 🎮 拼音大冒险

> 一个专为6岁儿童设计的拼音闯关学习网页游戏
> 支持上海版和人教版拼音教材

## ✨ 功能特点

### 🎯 核心学习功能
- **闯关模式**：8-12个关卡，难度递进，解锁机制
- **自由学习**：4种游戏化学习方式
  - 🗺️ 拼音探险 - 探索所有拼音卡片
  - ⚡ 闪电答题 - 限时挑战答题
  - 🧠 记忆翻牌 - 配对记忆游戏
  - ✍️ 拼音描红 - 跟读练习
- 🔊 **语音朗读**：使用浏览器TTS自动朗读拼音（支持中文发音）
- 🏆 **成就系统**：星星评级，进度追踪，成就解锁

### 🏠 宠物养成系统（新增）
- **宠物小屋**：领养可爱宠物（🐱→🐯→🦁 三阶段进化）
- **互动养成**：喂食、玩耍、升级
- **状态管理**：心情、饱食度、经验值系统
- **进化系统**：宠物随等级进化形态

### ⚔️ 宠物对战系统（新增）
- **技能商店**：6种技能可学习（利爪、撕咬、火焰、冰冻、雷电等）
- **武器商店**：4种装备可购买（钢爪、项圈、铠甲、王冠）
- **对战地图**：8种敌人挑战（小老鼠→巨龙）
- **回合制战斗**：宠物 vs 野生怪物

### 🎁 宝箱抽奖系统（新增）
- **钥匙获取**：每通关获得1把钥匙
- **随机奖励**：金币、宠物食物、经验值
- **开箱动画**：奖励弹窗庆祝效果

### 🛋️ 小屋装饰系统（新增）
- **装饰商店**：6种家具可购买（床、沙发、盆栽、台灯、地毯、挂画）
- **可视化小屋**：实时展示已购买的家具摆放
- **房间场景**：地板、墙壁、窗户背景

### 👨‍👩‍👧 家长后台
- **学习统计**：总星星、金币、关卡进度
- **金币修改**：家长可直接调整金币数量
- **进度查看**：每关完成情况和星级
- **密码保护**：默认密码 1234

## 📖 教材版本

### 上海版（8关）
1. 单韵母乐园（a o e i u ü）
2. 声母城堡（b p m f d t n l）
3. 声母王国（g k h j q x）
4. 声母世界（z c s zh ch sh r）
5. 复韵母天地（ai ei ui ao ou iu）
6. 鼻韵母星球（an en in un ün ang eng ing ong）
7. 整体认读岛（zhi chi shi ri zi ci si yi wu yu）
8. 整体认读王国（ye yue yuan yin yun ying）

### 人教版（12关）
- 分阶段学习声母、韵母、整体认读音节
- 更细致的关卡划分

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html` 或访问在线版本
2. 点击「开始冒险」进入主菜单
3. 选择「闯关模式」或「自由学习」开始学习
4. 进入「宠物小屋」体验养成和对战
5. 家长可点击「家长入口」查看进度（密码：1234）

## 🛠 技术栈

- HTML5 + CSS3 + JavaScript（纯前端，无需服务器）
- Web Speech API 语音合成（支持中文）
- LocalStorage 本地存储进度、金币、宠物数据
- 响应式设计，适配多种设备（优先iPad）

## 🔧 自定义配置

### 修改家长密码
编辑 `js/app.js` 中 `checkParentPassword` 函数：
```javascript
if (password === '1234') {  // 修改这里的密码
```

### 添加新的拼音内容
编辑 `js/pinyin-data.js`，在对应版本的 `levels` 数组中添加关卡。

### 调整语音速度
在设置界面可以调整，或修改 `js/game.js` 中的 `speechRate` 默认值。

## 📝 使用说明

### 儿童使用
1. 点击「开始冒险」→ 选择「闯关模式」或「自由学习」
2. **闯关模式**：听发音，选择正确的拼音，获得星星和金币
3. **自由学习**：选择游戏模式，边玩边学
4. **宠物小屋**：
   - 点击「宠物对战」→ 学习技能 → 挑战敌人
   - 点击「我的房间」查看装饰效果
   - 点击「装饰商店」购买家具
5. 使用金币在商店购买技能和装备

### 家长使用
1. 在启动画面点击「家长入口」
2. 输入密码（默认：1234）
3. 查看学习进度、统计数据
4. 点击金币旁的「修改」按钮可调整金币数量
5. 查看每关完成情况和星级评定

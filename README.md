# 🎮 拼音大冒险

> 一个专为6岁儿童设计的拼音闯关学习网页游戏
> 支持上海版和人教版拼音教材

## ✨ 功能特点

- 🎯 **闯关模式**：8-12个关卡，难度递进，解锁机制
- 📚 **自由学习**：随时查阅所有拼音，点击发音
- 🔊 **语音朗读**：使用浏览器TTS自动朗读拼音
- 🏆 **成就系统**：星星评级，进度追踪
- 👨‍👩‍👧 **家长后台**：查看学习进度和统计数据
- 📱 **多设备适配**：优先iPad，支持电脑和手机

## 📖 教材版本

### 上海版（8关）
1. 单韵母乐园（a o e i u ü）
2. 声母城堡（b p m f d t n l）
3. 声母王国（g k h j q x）
4. 声母世界（z c s zh ch sh r）
5. 复韵母天地（ai ei ui ao ou iu）
6. 鼻韵母星球（an en in un ün ang eng ing ong）
7. 整体认读岛（zhi chi shi ri zi ci si yi wu yu）
8. 整体认读王国（ye yue yuan yin yun ying）

### 人教版（12关）
- 分阶段学习声母、韵母、整体认读音节
- 更细致的关卡划分

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html`
2. 点击「开始冒险」进入主菜单
3. 选择「闯关模式」开始学习
4. 家长可点击「家长入口」查看进度（密码：1234）

## 🛠 技术栈

- HTML5 + CSS3 + JavaScript（纯前端，无需服务器）
- Web Speech API 语音合成
- LocalStorage 本地存储进度
- 响应式设计，适配多种设备

## 🔧 自定义配置

### 修改家长密码
编辑 `js/app.js` 第 235 行：
```javascript
if (password === '1234') {  // 修改这里的密码
```

### 添加新的拼音内容
编辑 `js/pinyin-data.js`，在对应版本的 `levels` 数组中添加关卡。

### 调整语音速度
在设置界面可以调整，或修改 `js/game.js` 中的 `speechRate` 默认值。

## 📝 使用说明

### 儿童使用
1. 点击「开始冒险」→「闯关模式」
2. 选择要学习的关卡
3. 听发音，选择正确的拼音
4. 答对获得星星，答错会扣星
5. 完成关卡解锁下一关

### 家长使用
1. 在启动画面点击「家长入口」
2. 输入密码（默认：1234）
3. 查看学习进度、统计数据
4. 设置每日学习目标

# 拼音游戏升级版

## 项目结构
```
pinyin-game/
├── assets/          # 视觉资源
│   ├── characters/  # 角色立绘
│   ├── scenes/      # 场景背景
│   └── ui/          # UI组件
├── src/
│   ├── components/  # 游戏组件
│   ├── scenes/      # 游戏场景
│   ├── utils/       # 工具函数
│   └── data/        # 拼音数据
├── index.html
└── styles.css
```

## 核心功能
1. 认知脚手架学习系统
2. 自适应难度调整
3. 多模态学习体验
4. 家长深度参与

## 技术栈
- HTML5 + CSS3 + JavaScript
- 通义万相API（视觉资源）
- 讯飞/百度语音API（语音识别）
